# CLAUDE.md — Accounts

## Purpose

Canonical registry of external service accounts used across the different entities (weibeld, W5D Labs, Rye & Cheese, w5d.io). Intended as a programmatic reference for platform systems that need non-secret identifiers.

## Contents

`accounts.yaml` contains login credentials (email, account/org name) and service-specific non-secret identifiers (e.g. account IDs, project IDs) for each service. Secrets (API keys, passwords) are never stored here.
